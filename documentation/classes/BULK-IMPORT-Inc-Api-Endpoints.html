register_api_endpoints()

class-api-endpoints.php : 48

Register API Endpoints.

public register_api_endpoints() : mixed

Defines custom REST API routes and their corresponding callbacks.

Endpoints:

  1. /server-status: Returns the server status to ensure API connectivity.
  2. /delete-products: Deletes all WooCommerce products permanently.
  3. /delete-trash-products: Permanently deletes all trashed WooCommerce products.
  4. /delete-woo-cats: Deletes all WooCommerce product categories.
  5. /test-job: Simulates a test job by logging a unique identifier.
