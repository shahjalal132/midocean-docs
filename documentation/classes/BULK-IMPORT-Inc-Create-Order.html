Creates and processes an order by interacting with an external API.

public create_order(int $order_id) : void

This method is triggered after an order is placed in WooCommerce. It performs the following steps:

  1. Retrieves order details.
  2. Sends the order details to an external API.
  3. Processes the API response to extract specific details such as order type and printing positions.
  4. Saves the extracted data to order meta in WooCommerce for future reference.
  5. Deletes cookies related to the order after the process is complete to ensure data cleanup.
Parameters
$order_id : int

The ID of the WooCommerce order being processed.

public update_cart_item_price(WC_Cart $cart) : void

This function modifies the price of items in the WooCommerce cart based on a calculated value retrieved from a cookie (_calculated_price). It ensures the new price is applied without causing conflicts during the calculation of totals.

Parameters
$cart : WC_Cart

The WooCommerce cart object containing all cart items and associated data.

Return values
void —

How it works:

  1. The function hooks into the woocommerce_before_calculate_totals action, which is triggered before the cart totals are calculated.
  2. To prevent multiple executions of this logic (which could cause unintended behavior), it uses did_action to ensure the code runs only once.
  3. For each item in the cart:
    • The product ID is retrieved.
    • The function checks if the _calculated_price cookie exists and retrieves its value.
    • The product's price in the cart is updated with the new price from the cookie.
  4. This allows dynamic price adjustments, such as those based on custom calculations or user actions, to be reflected in the cart and checkout.
